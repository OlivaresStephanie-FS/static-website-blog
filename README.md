# Memory Lane

## Overview
Memory Lane is a static blog-style website built with Next.js and deployed through Vercel. The project functions as a simple developer journal focused on documenting lessons learned throughout web development studies.

## Features
- Homepage with custom layout
- Dedicated posts archive page
- Dynamic individual post pages
- Markdown-based blog content
- Custom CSS styling
- Navigation between pages
- Static deployment workflow using GitHub and Vercel

## Technologies Used
- Next.js
- React
- TypeScript
- Markdown
- CSS
- GitHub
- Vercel
