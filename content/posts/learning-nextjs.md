Building this project with Next.js gave me more experience working with React-based applications and file-based routing. I liked how organized the project structure felt compared to some smaller JavaScript projects I worked on previously.

One of the most interesting parts for me was seeing how pages are automatically generated based on folders and file names. It made navigation and page structure feel much easier to manage while still keeping the project clean and scalable.

This project also helped reinforce the connection between development, GitHub, and deployment. Instead of only building locally, I was able to see how modern front-end projects move from code editor to live website through a full workflow.
